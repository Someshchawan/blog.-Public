# Somesh Chawan — Blog

Personal technical blog writing about AI documentation, developer experience, and building tools that make technical content trustworthy.

**Live site:** [someshchawan.github.io/blog](https://someshchawan.github.io/blog)

## Posts

- **How a Custom Gemini Gem Changed My Documentation Workflow** — What happened when I stopped treating AI as an experiment and started building it into my daily process.
- **What I Learned Building an Evaluation Layer for AI-Generated Content** — The model said "your account shows two transactions." It was lying. Here is how I built the system that catches that.

## Part of a portfolio

This blog is one of three connected projects:

1. **[AI Support Copilot](https://github.com/Someshchawan/ai-support-copilot)** — The production AI system with evaluation layer
2. **[Documentation Site](https://github.com/Someshchawan/ai-support-copilot-docs)** — Deployed Docusaurus site with CI/CD
3. **This blog** — Writing about the thinking behind both projects

## Tech stack

- Jekyll 4 on GitHub Pages
- Custom design (no theme dependency)
- Deployed automatically on push to main

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Site runs at `http://localhost:4000/blog/`
